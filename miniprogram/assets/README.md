# Assets 资源目录

## 图标文件说明

本目录需要放置以下图标文件（建议尺寸 32x32 或 48x48 px）：

### TabBar 图标
- `briefcase.png` - 岗位图标（未选中）
- `briefcase-active.png` - 岗位图标（选中，紫色）
- `calendar.png` - 面试图标（未选中）
- `calendar-active.png` - 面试图标（选中，紫色）
- `book.png` - 面经图标（未选中）
- `book-active.png` - 面经图标（选中，紫色）
- `user.png` - 我的图标（未选中）
- `user-active.png` - 我的图标（选中，紫色）

### 页面图标
- `search.png` - 搜索图标
- `location.png` - 位置图标
- `salary.png` - 薪资图标
- `calendar.png` - 日期图标
- `add-white.png` - 添加按钮图标（白色）
- `empty.png` - 空状态图标

## 临时方案

目前可以使用字体图标（IconFont）或在线图标库替代，
后续将替换为正式的图标资源。

## 图标设计规范

- **颜色**：未选中状态使用 #9CA3AF，选中状态使用 #8B5CF6
- **尺寸**：TabBar 图标建议 48x48 px，页面图标建议 32x32 px
- **格式**：PNG 格式，支持透明背景
