# 求职追踪助手 AI版 - 可交互原型

这是一个完整的可交互UI原型，展示求职追踪助手的所有功能和界面。

## 如何使用

### 方法一：直接在浏览器中打开

1. 双击 `index.html` 文件，或右键选择"打开方式" → 选择浏览器

### 方法二：使用本地服务器（推荐）

```bash
# 进入原型目录
cd prototype

# 使用Python启动服务器
python3 -m http.server 8000

# 或使用Node.js
npx http-server -p 8000

# 然后在浏览器访问
# http://localhost:8000
```

## 设计规范

### 色彩系统
- **主色调**：蓝色 #0369A1（常规功能）
- **AI主题色**：紫色渐变 #6366F1 → #8B5CF6
- **成功色**：#10B981
- **警告色**：#F59E0B
- **错误色**：#EF4444

### AI元素特征
- 紫色渐变背景
- 左侧紫色边框（4px）
- 浅紫色卡片背景
- 🤖 AI图标
- ✨ 思考动画（三个跳动的圆点）

### 交互反馈
- Toast提示（操作成功/失败）
- Modal弹窗（确认、提示）
- 加载动画（AI处理中）
- 点击反馈（卡片缩放）

## 技术实现

### 技术栈
- **HTML5** - 页面结构
- **TailwindCSS** - 样式框架（CDN引入）
- **原生JavaScript** - 交互逻辑
- **Font Awesome** - 图标库（CDN引入）

### 架构设计
- **单页应用（SPA）** - 使用JavaScript控制页面渲染
- **组件化渲染** - 每个页面对应一个渲染函数
- **模拟数据** - 内置模拟数据，无需后端
- **路由管理** - 简单的状态路由系统

### 文件结构
```
prototype/
├── index.html          # 主入口文件
├── css/
│   └── common.css      # 公共样式
├── js/
│   ├── data.js         # 模拟数据
│   ├── common.js       # 公共脚本（路由、Toast、Modal、AI）
│   └── app.js          # 主应用逻辑（页面渲染、事件处理）
└── README.md           # 本文件
```

## 注意事项

1. **仅为原型**：当前是UI原型，数据存储在内存中，刷新后会重置
2. **模拟AI**：AI功能为模拟响应，实际使用需要接入真实AI API
3. **浏览器兼容**：建议使用现代浏览器（Chrome、Edge、Safari）
4. **移动端适配**：最佳体验在手机屏幕或移动端模式下
